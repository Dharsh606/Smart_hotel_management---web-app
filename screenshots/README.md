# Screenshots Directory

Place your application screenshots in this directory.

## Required Screenshots:

1. **login.png** - Login page screenshot
2. **dashboard.png** - Admin dashboard screenshot
3. **booking.png** - Booking form screenshot
4. **rooms.png** - Rooms overview screenshot
5. **checkin.png** - Check-in/Check-out process screenshot
6. **logs.png** - Activity logs page screenshot

## Screenshot Guidelines:

- Use PNG format for best quality
- Recommended resolution: 1920x1080 or higher
- Ensure screenshots are clear and showcase the UI effectively
- Crop to focus on the main content area
